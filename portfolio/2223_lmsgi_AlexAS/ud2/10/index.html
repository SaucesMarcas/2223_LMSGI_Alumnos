Aprendiendo Java

Para poder programar en Java necesitaremos instalar un JDK, este lenguaje es fuertemente tipado, por lo que deberemos asegurarnos de escribir el codigo correctamente, respetando mayusculas y minusculas y usando el punto y coma, de no hacerlo, el programa dara error
por ejemplo si escribimos System.out.println("X") escribira X, de escribir system.out.println("X") no funcionara
